Unity 2D로 제작한 Bank System 포트폴리오이다. 이 게임은 Json Utility 등의 기술을 사용하여 제작하였으며, 사용자의 정보를 저장하는 기능을 제작하였다. 
해당 시스템에서는 회원가입과 로그인, 입금, 출금, 송금 등의 기능을 제공하고 있으며, 자세한 것은 동영상을 통해 확인할 수 있다.
